# Umfragetool

Ein einfaches Umfrage-Verwaltungstool mit Express.js und SQLite.

## Installation

```bash
npm install
```

## Starten

```bash
npm start
```

Die App läuft dann auf `http://localhost:3000`

## Features

- Umfragen erstellen und verwalten
- Antworten speichern
- SQLite Datenbank

## Technologie

- Node.js / Express.js
- SQLite3
- HTML/CSS/JavaScript (Frontend)
